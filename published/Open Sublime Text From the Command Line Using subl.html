Open Sublime Text From the Command Line Using subl.exe (Windows)
Adding Sublime to Your PATH
A nice shortcut to do this is to open up your command line and typing sysdm.cpl. Then you should create a New System Varible and add to your PATH:
subl.exe - open file / folder Sublime in new / same Window
Just find the one you want and type: [bash] subl.exe file_name #or subl.exe folder_name #in current sublime window subl.exe file_name --add #or subl.exe folder_name --add [/bash]
